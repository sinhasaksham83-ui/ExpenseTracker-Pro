# 💰 ExpenseTracker Pro

A modern and responsive personal finance management web application built using **HTML, CSS, and JavaScript**.

ExpenseTracker Pro helps users manage income, expenses, analyze spending patterns, and maintain better financial control through an interactive dashboard.

- AI-style spending insights
